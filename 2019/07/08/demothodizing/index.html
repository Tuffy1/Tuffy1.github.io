一 含义

函数调用

  1. Object.prototype.toStrig: 表示 Object 对象的 toString 方法；

  2. bind：是 Function.prototype 的一个方法，该方法会有创建一个新函数，并将该函数的 this 指向其参数，并返回这个函数；

  3. call：是 Function.prototype 的一个方法，与 bind 不同，call 函数会直接执行，并将函数内 this 指向参数；
